## 프로잭트 소개
- 지역기반 헬스 커뮤니티 플랫폼


## 기능 설명
 - 모션인식을 이용한 운동 챌린지
 - 헬스용품 거래 서비스
 - 운동 지식인 점수 조회 및 검색

## 컴퓨터 구성 / 필수 조건 안내 (Prerequisites)
* ECMAScript 6 지원 브라우저 사용
* 권장: Google Chrome 버젼 77 이상

## 기술 스택 (Technique Used) 
### Server(back-end)
 - Python 
 
### Front-end
 -  html/css/js

## 설치 안내 (Installation Process)
```bash
$ git clone https://github.com/2022-oasis-hackathon/japda.git
```
